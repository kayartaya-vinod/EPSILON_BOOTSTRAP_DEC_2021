First HTML with bootstrap CSS

Different sizes of devices

  • x-small (xs): <576px
  • small (sm): >=576px and <768px
  • medim (md): >=768px and <992px
  • large (lg): >=992px and <1200px
  • x-large (xl): >=1200px and <1400px
  • xx-large (xxl): >=1400px

.container class and the width allotted:

  • xs: 100%
  • sm: 540px
  • md: 720px
  • lg: 960px
  • xl: 1140px
  • xxl: 1320px

Other .container class variants:

  • .container-sm
  • .container-md
  • .container-lg
  • .container-xl
  • .container-xxl

.container-fluid

takes 100% width irrespective of devices.
